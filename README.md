# UrbanEye AI

AI-Based Smart Traffic Density and Civic Issue Management System

## Project Overview
UrbanEye AI is an intelligent system designed to:
- Monitor traffic density using AI-based vehicle detection
- Classify traffic levels (Low, Moderate, Heavy)
- Detect civic infrastructure issues such as potholes and garbage overflow
- Generate structured digital complaints
